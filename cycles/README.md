# Cycles

Jeder 6-Monats-Lauf bekommt einen **Codename** und einen eigenen Ordner.

## Codename-Regeln

| ✅ Gut | ❌ Schlecht |
|--------|------------|
| `projekt-linde` | `ai-hr-tool` |
| `projekt-ufer` | `saas-idee-2026` |
| `projekt-kasten` | Echter Produktname |

**Warum:** Öffentliche Dokumentation ohne Idee zu verraten (L3).

## Neuen Cycle anlegen

```bash
mkdir -p cycles/projekt-linde
cp templates/cycle-log-vorlage.md cycles/projekt-linde/README.md
```

Dann [templates/codename-cycle-start.md](../templates/codename-cycle-start.md) durchgehen.
