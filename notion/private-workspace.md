# Privater Notion-Workspace

Volle Daten bleiben hier. Nur kuratierte Ausschnitte gehen nach GitHub (`cycles/`) und LinkedIn.

---

## Cycle anlegen (wenn Idee vorhanden)

1. Dupliziere **„[Template] Neuer Cycle"** (s. [private-cycle-template.md](./private-cycle-template.md))
2. Benenne um: `Projekt [Codename]`
3. Fülle **Idee (intern)** sofort aus — das bleibt privat
4. Lege öffentliches Log an: `cycles/[codename]/README.md` auf GitHub

## Export-Queue

Vor jedem LinkedIn-Post prüfen:

| Post-Typ | Erlaubte Tiefe | Check |
|----------|----------------|-------|
| Monats-Log | L2 | Keine Namen, gerundete Zahlen |
| Problem-first | L3 | Kategorie, kein Produktname |
| Stop/Weitermachen | L4 | Entscheidung OK, Details intern |
| Pricing | L3 | Keine exakten Zahlen |

**Workflow:**
1. Entwurf in Export-Queue
2. L-Stufe markieren
3. Erst dann posten
